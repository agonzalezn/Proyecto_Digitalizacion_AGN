# Contributing to VoleyPro

Thank you for your interest in contributing to VoleyPro! We appreciate your support in making this open-source volleyball digitization tool even better. By contributing, you help improve the experience for coaches, players, and fans alike.

## 🛠️ How to Contribute

To ensure a smooth contribution process, please follow this GitHub workflow:

1. **Fork** the repository to your own GitHub account.
2. **Clone** the forked repository to your local machine.
3. **Create a branch** for your feature or bug fix (e.g., `git checkout -b feature/new-amazing-feature`).
4. **Make changes** to the codebase, ensuring your code is clean and well-structured.
5. **Document code**, especially new functions or complex logic. It is mandatory to use **JSDoc** for JavaScript documentation.
6. **Commit changes** using a clear, semantic commit message. For example: `feat: add player statistics export functionality` or `fix: resolve score calculation bug`.
7. **Push to branch** on your forked repository (`git push origin feature/new-amazing-feature`).
8. **Open a Pull Request** against the main repository branch, describing your changes in detail so they can be reviewed.

## 🚀 Future Roadmap (Ideas for contribution)

If you are looking for ideas on where you can contribute, here are some exciting features we plan to implement:

- **Cloud Integration**: Implement backend support using Firebase or Supabase to sync match data, user profiles, and team stats across multiple devices seamlessly.
- **PWA Support**: Transform VoleyPro into a Progressive Web App. This involves adding service workers and a `manifest.json` to allow offline functionality and installation on mobile devices.
- **Data Export Feature**: Allow users to export match statistics and game logs into various formats such as CSV, PDF, or Excel options for further analysis.
- **Advanced Match Settings**: Introduce a robust system for complex match scenarios, including a global Undo/Redo function (e.g., `Ctrl+Z`) to correct scoring mistakes effortlessly.

## 🤝 Code of Conduct

We are committed to providing a welcoming, inclusive, and inspiring community for all. Please be respectful, considerate, and professional in your interactions. Harassment or abusive behavior of any kind will not be tolerated. Keep discussions focused on the project and be open to constructive feedback.
